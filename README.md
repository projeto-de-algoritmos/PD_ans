## Instalação

**Linguagem**: C++ e Python<br>

Com o compilador G++/GCC instalado na máquina, compile e execute o arquivo da seguinte forma:

    g++ arquivo.cpp -o executavel
    ./executavel

    ou

    gcc arquivo.c -o executavel
    ./executavel

Ou, em python:

    python arquivo.py
